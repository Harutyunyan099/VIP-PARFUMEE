## Ֆայլերի կառուցվածքը (հարթ, առանց data.js)
```
sillage/
├── index.html      ← գլխավոր էջը
├── style.css       ← ամբողջ դիզայնը (գիշեր + ցերեկ)
├── app.js          ← ամբողջ տրամաբանությունը
├── parfumes.json   ← ՏՎՅԱԼՆԵՐԻ ՄԻԱԿ ԱՂԲՅՈՒՐԸ (նկարների լինկերը՝ այստեղ)
└── README.md
```
Տվյալներն ու նկարների լինկերը պահվում են **միայն** `parfumes.json`-ում՝ որպես օբյեկտներ։ data.js այլևս չկա — JSON-ը խմբագրում ես, կայքն ինքնաբերաբար թարմանում է։

## ⚠ Ինչպես բացել (կարևոր)
Քանի որ տվյալները միայն JSON-ում են, browser-ները կարդում են այն ցանցային հարցումով.
- **Լավագույն ձև՝** VS Code → Live Server, կամ թերմինալում `npx serve` / `python -m http.server` → բացիր http://localhost:…
- **Firefox / Safari**՝ սովորաբար աշխատում է նաև index.html-ի կրկնակի սեղմումով
- **Chrome**-ը file:// ռեժիմում արգելում է JSON-ի ընթերցումը — այդ դեպքում կայքը ցույց կտա բարեկիրթ հաղորդագրություն՝ հրահանգներով (չի «ջարդվում»)

## 🖼 Նկարները — քո JSON-ի լինկերն են, 3-փուլ բեռնումով
1. Բնօրինակ լինկը (`referrerpolicy="no-referrer"`)
2. Չստացվեց (fimgs.net-ի hotlink արգելք) → **images.weserv.nl** proxy — նկարները երաշխավորված երևում են
3. Ինտերնետ չկա → ոսկե շշի placeholder
Բեռնվելիս՝ fade-in + shimmer skeleton։ PNG-ները՝ մաքուր, JPG-ները՝ ոսկե photo-շրջանակում։
*(Black Orchid-ի (id:1) դատարկ image դաշտը վերականգնվել է քո առաջին JSON-ի լինկով)*

## 🎬 Cinema — scroll-բացում (նոր)
Marquee-ից անմիջապես հետո՝ 3,6 էկրան բարձրությամբ sticky տեսարան.
1. **Black Orchid-ի իսկական ոճով սև տուփ** (TOM FORD · BLACK ORCHID ոսկե տառերով) — կափարիչը բարձրանում-հեռանում է
2. Շիշը (քո JSON-ի PNG-ն) **մեծ չափով** դուրս է բարձրանում՝ ոսկե glow-ով
3. **Ցողում**՝ 53 տարր երեք շերտով — ոսկե կաթիլներ, պտտվող ✦ փայլեր, լույսի շիթեր + մառախուղ ու պտտվող haze; կաթիլներն ունեն թեթև «ձգողականության» աղեղ
4. Finale քարտ → «Բացահայտել այս բույրը» → բացում է modal-ը
Շարժումը inertia-smoothing-ով է (ֆիզիկային մոտ սահունություն)։ Reduced-motion → ստատիկ կադր։
